--Best Practice means Using CSS3 (Cascading Style Sheets)--

There are many benefits to using CSS3.

Web Developers use CSS3, the current version of Cascading Style Sheets, to create a master style sheet that can be applied to all the web pages within any web design project. Utilizing one main style sheet to implement a consistent style to all of the html pages with in a project(website) provides two main benefits:
First, if there are improvements to be made or bugs to be fixed, you only need to make the change to one document (the CSS style sheet) and all pages on the site will reflect the change as well.
Second, the fluidity that is created by applying the same style to all of the pages with in a project "pulls everything together". This helps to ensure a visualy pleasing experience for the user. Also, as a result of the consistent stlying, connection between the HTML pages is re-inforced and strengthened creating a cohesive presentation of information.

The main "take-away" here is: Not only is using CSS3 considered "Best Practice";...It is also true that utilizing cascading style sheets (CSS3) in your workflow is the most efficient way to apply aesthetic code to your web projects.
